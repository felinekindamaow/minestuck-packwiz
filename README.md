# Minestuck (for Packwiz)

An alternative solution to installing the [Minestuck server pack](https://minestuck.com/server)
using [Packwiz](https://github.com/packwiz/packwiz) and the MultiMC or Prism launchers.

## Installation

1. [Retrieve the latest MultiMC/Prism instance from the GitHub releases](https://github.com/felinekindamaow/minestuck-packwiz/releases/latest)
2. In MultiMC/Prism, click `Add instance` in the top-left corner of the main window.
3. Enter the `Import` tab and select the `.zip` file you downloaded.
